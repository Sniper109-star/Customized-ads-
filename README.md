# Ad Control

Run ads on Facebook and TikTok from one dashboard.

## Features

- Campaign and ad management UI
- Facebook Ads and TikTok Ads integrations
- SQLite persistence with Drizzle ORM
- Authentication
- Standard UI component library

## Getting Started

```bash
bun install
bun dev
```

Open http://localhost:3000.

## Scripts

| Command | Description |
|---------|-------------|
| `bun dev` | Start development server |
| `bun build` | Build for production |
| `bun start` | Start production server |
| `bun lint` | Run ESLint |
| `bun typecheck` | Run TypeScript checks |

## Tech Stack

- Next.js 16 + React 19
- TypeScript + Tailwind CSS 4
- Drizzle ORM + better-sqlite3
- Class Variance Authority + Tailwind Merge
